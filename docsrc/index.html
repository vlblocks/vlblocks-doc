Blocks is an open source modular MATLAB framework which allows the user to avoid needlessly repeating computation.

Download Blocks 0.1 (Windows, Mac, Linux)

Blocks may be easily used for your own experiments, and comes with some useful modules for computer vision experiments: an implementation of a simple image level bag-of-features (BoF) classifer, a fast agglomerative information bottlneck (AIB) sliding window localization scheme, and a class segmentation scheme using superpixel neighborhoods and a conditional random field (CRF). It also provides support for running experiments on clusters of compuers.
